
# ✨ Advanced Text Style Generator

Transform your plain text into stylish Unicode text with ease!  
This project is a feature-rich, browser-based tool for generating fancy, cool, and copyable Unicode text in real-time. Whether you need aesthetic fonts for social media, stylish bios, or decorative messages — this tool has got you covered!

Link https://harryme17.github.io/Advanced-Text-Style-Generator/



## 🚀 Features

-   🔤 **50+ Stylish Unicode Font Transformations**  
    From bold, italic, cursive, serif, and fraktur to fun emoji, square, circled, mirrored, and more!
    
-   ❤️ **Favorites System**  
    Mark and save your favorite styles with one click. Automatically saved using `localStorage`.
    
-   🎨 **Custom Style Editor**  
    Create and apply your own text styles using combinations like bold, italic, underline, etc.
    
-   📋 **Copy with One Click**  
    Each stylized text block includes a handy copy button.
    
-   🔍 **Live Search Filter**  
    Quickly find your favorite style by typing keywords.
    
-   💾 **Persistent Local Data**  
    Your favorites and text input are stored locally — even after page refresh.
    
-   📱 **Fully Responsive UI**  
    Clean and modern glassmorphism-inspired design optimized for both desktop and mobile.
    
-   💡 **Notification System**  
    Smooth feedback for actions like copy, favorite added/removed, and errors.
